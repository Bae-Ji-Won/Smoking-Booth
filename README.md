## ☁️ 프로젝트 주요 기능 
#### &nbsp;&nbsp; 1. 버튼을 누르면 출입구 Open 
#### &nbsp;&nbsp; 2. 사람이 들어오면 LCD로 인원수 출력(1명 이상일시 자동으로 부스안 조명 켜짐)
#### &nbsp;&nbsp; 3. 문이 제대로 닫히지 않았을 경우 LED 점등
#### &nbsp;&nbsp; 4. 연기가 감지되면 환풍기 작동
